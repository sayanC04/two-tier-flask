# 🚀 Flask-MySQL Two-Tier App

A streamlined 2-tier architecture demonstrating container communication between a **Flask** backend and a **MySQL** database.

---

## 🛠️ Tech Stack
- **Backend:** Flask (Python)
- **Database:** MySQL 5.7
- **Orchestration:** Docker & Docker Compose

---

## ⚡ Quick Start

### 1. Environment Setup
Create a `.env` file in the root directory:

DB_NAME=mydb
DB_USER=root
